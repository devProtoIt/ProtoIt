Beschrijving:
Dit apparaat leest het id van een rfid-tag die voor de sensor wordt gehouden. Een id bestaat uit een serie van acht bytes, die hexadecimaal worden geschreven, bijvoorbeeld "FE 83 00 61 A4 8C 10 03". Er is verder geen gegevensuitwisseling. De meest voorkomende toepassing is toegangscontrole.

Onderdelen:
RFID sensor

ProtoIt

Instellingen:
PoortOm met de rfid-sensor te kunnen werken, moet je de Poort invullen. Dit is het nummer van de poort op de Nxt Brick, waar je de sensor op aansluit (1, 2, 3 of 4).

ID tag 1Geef het id van de tag die als nummer 1 zal worden herkend.
ID tag 2Geef het id van de tag die als nummer 2 zal worden herkend.
ID tag 3Geef het id van de tag die als nummer 3 zal worden herkend.

Zendt de signalen:
Tag-IDBevat het id van de tag die voor de sensor wordt gehouden. Het signaal is leeg wanneer geen tag wordt herkend.
Is tag 1Dit signaal treedt op als tag nummer 1 (zie de Instellingen) voor de sensor wordt gehouden.
Is tag 2Dit signaal treedt op als tag nummer 2 (zie de Instellingen) voor de sensor wordt gehouden.
Is tag 3Dit signaal treedt op als tag nummer 3 (zie de Instellingen) voor de sensor wordt gehouden.
