Lamp

Platform: ARDUINO

Beschrijving:
Een lamp (of led) wordt aan of uit geschakeld. Ook kun je hem laten knipperen. Stel de tijd in dat hij aan en uit blijft.
Eventueel kun je de lamp via ´Pwm´ harder of zachter laten branden. ´Pwm´ staat voor pulse-wide-modulation. Je moet hem dan wel op een speciale ´Pwm´-pin aansluiten.

LET OP! Je mag een lamp niet rechtstreeks op de Arduino-pinnen aansluiten. Dan gaat de Arduino stuk.

Instellingen:
PinHet pin-nummer van de Arduino waarop je de lamp aansluit.

Op de programmategel:
Helderheid (%)Een waarde van 0 tot 100, die de lichtsterke van de lamp bepaalt.
Standaard ingesteld op 100%.
Tijd aan (msec)De tijd in milliseconden dat de lamp aan blijft.
Standaard op 0 ingesteld (= altijd aan).
Tijd uit (msec)De tijd in milliseconden dat de lamp uit blijft.
Standaard op 0 ingestald (= altijd aan).
Wanneer aan?Als het ingevulde signaal optreedt, zal de lamp aan gaan.
Standaard op het signaal Direct ingesteld.
Wanneer uit?Als het ingevulde signaal optreedt, zal de lamp uit gaan.
Standaard op het signaal Nooit ingesteld.
