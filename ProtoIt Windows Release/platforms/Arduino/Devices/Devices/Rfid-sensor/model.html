Beschrijving:
Een RFID-sensor werkt samen met ID-kaarten (ook wel tags genoemd). De sensor leest de kaart en kan de ID weergeven.

Instellingen:
Mosi-pinDe Arduino-pin waarop de MOSI-pin van de sensor is aangesloten.
Miso-pinDe Arduino-pin waarop de MOSI-pin van de sensor is aangesloten.
CS-pinDe Arduino-pin waarop de CS-pin van de sensor is aangesloten.
SCK-pinDe Arduino-pin waarop de SCK-pin van de sensor is aangesloten.

Zendt de signalen:
ID GelezenHet signaal ID Gelezen wordt uitgezonden nadat een ID-kaart wordt gepresenteerd.
Kaart-IDBevat de ID van de laatst gelezen ID-kaart.
