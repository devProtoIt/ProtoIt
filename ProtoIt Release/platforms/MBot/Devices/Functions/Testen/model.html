Beschrijving:
De tile Testen ontvangt twee signalen om door te geven. Een testsignaal kiest welk van de twee wordt doorgegeven. Het doorgegeven signaal wordt uitgezonden als signaal Actie.

Op de programmategel:
TestsignaalDit signaal wordt getest. Wanneer het wordt uitgezonden is de test "gelukt" en zoniet, dan is de test "mislukt".
Als gelukt: ZendWanneer de test is "gelukt" wordt dit signaal verzonden via het signaal Actie.
Als mislukt: ZendWanneer de test is "mislukt" wordt dit signaal verzonden via het signaal Actie.

Zendt de signalen:
ActieBevat het signaal dat moet worden uitgezonden bij een "gelukte" of "mislukte" test.
